# Retail Sales Data Analysis

## Project Overview

This project analyzes a retail sales dataset to understand sales performance, product category performance, and regional performance.

The goal is to clean the data, perform exploratory data analysis, create visualizations, and extract business insights that can support decision-making.

## Tools Used

- Python
- Pandas
- Matplotlib
- Jupyter Notebook
- GitHub

## Dataset

The dataset contains retail sales records with columns such as:

- Date
- Category
- Sales
- Quantity
- Profit
- Region

## Project Steps

1. Loaded the dataset using Pandas
2. Checked dataset shape, column names, data types, and missing values
3. Converted the Date column to datetime format
4. Converted Sales, Quantity, and Profit columns to numeric format
5. Removed missing values and duplicates
6. Created useful columns such as Year, Month, Month Name, Day of Week, Sales per Item, Profit Margin, Sales Level, and Profit Status
7. Performed exploratory data analysis
8. Created visualizations
9. Summarized key business insights and recommendations

## Business Questions Answered

### Sales Performance

- What are total sales by month?
- Which day of the week has the highest sales?
- What is the sales trend over time?

### Product Insights

- Which product category generates the most revenue?
- Which product category has the highest quantity sold?

### Regional Performance

- Which region has the highest sales?
- Which region generates the highest profit?

## Key Insights

- The project identifies the strongest sales periods.
- The analysis shows the best-performing product categories by revenue and quantity sold.
- Regional analysis helps compare sales and profit across different regions.
- Profit margin analysis helps understand profitability, not only revenue.

## Recommendations

- Focus marketing efforts on the best-performing product categories.
- Monitor high-quantity categories to ensure enough inventory is available.
- Study top-performing regions to understand what drives stronger performance.
- Investigate lower-performing regions to improve sales and profitability.
- Track monthly sales trends to prepare inventory and promotions before high-sales periods.

## Files

- `retail_sales_analysis.ipynb`: Jupyter Notebook containing data cleaning, EDA, visualizations, and insights
- `cleaned_retail_sales.csv`: Cleaned dataset used for analysis

This project demonstrates a complete beginner-friendly data analysis workflow: data loading, cleaning, exploratory data analysis, visualization, and business insight generation.
